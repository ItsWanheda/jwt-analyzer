# --- Testing ---
# The test framework. pytest is non-negotiable - we use fixtures, parametrize,
# and the assertion rewriting magic.

# Coverage reporting. Shows which lines your tests don't touch.
# Run: pytest --cov=core --cov=utils --cov-report=term-missing

# Cleaner mocking. Wraps unittest.mock with a pytest fixture API.
# Useful for mocking out jwt.decode, HTTP calls, file I/O in tests.

# Mock HTTP responses without making real network calls.
# Critical for testing core/jwks.py without hitting real auth servers.

# Mock datetime/time.time() so expiry tests are deterministic.
# Otherwise your tests break every time the clock ticks.

# Generate test data: fake JWTs, keys, etc.
